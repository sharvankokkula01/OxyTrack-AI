# OxyTrack AI

A real-time Oxygen Cylinder Monitoring and Management System built using React, TypeScript, Supabase, and Tailwind CSS.

## Features

- Real-time oxygen cylinder monitoring
- Dashboard analytics
- Cylinder inventory management
- Alert and notification system
- Device management
- Maintenance tracking
- Hospital-wide oxygen monitoring
- Supabase cloud database integration

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Supabase
- Vite
